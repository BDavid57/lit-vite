# 🚀 Lit + Vite Project

A modern, fast, and lightweight web application built using **Lit** for building Web Components and **Vite** as the development and build tool.

## ✨ Overview

This project demonstrates how to build scalable and performant web interfaces using native Web Components powered by **Lit**, combined with the ultra-fast development experience provided by **Vite**.

It focuses on simplicity, reusability, and performance without relying on heavy frameworks.

## 🧱 Tech Stack

* ⚡ **Vite** – Fast development server and optimized production builds
* 🧩 **Lit** – Lightweight library for building Web Components
* 🟦 **TypeScript** – Type-safe JavaScript development
* 🌐 **Web Components** – Encapsulated, reusable UI elements

## 📦 Features

* ⚡ Instant hot module replacement (HMR) via Vite
* 🧩 Reusable Lit components with reactive properties
* 🎯 Encapsulated styling using Shadow DOM
* 📱 Responsive and modular UI structure
* 🧪 Easy local development setup
* 🚀 Optimized production builds

## 📁 Project Structure

```
src/
  components/     # Reusable Lit components
  main.ts         # App entry point
index.html        # Vite entry HTML
vite.config.ts    # Vite configuration
```

## 🚀 Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Start development server

```bash
npm run dev
```

### 3. Build for production

```bash
npm run build
```

### 4. Preview production build

```bash
npm run preview
```

## 🧠 Why Lit + Vite?

* Lit gives you **native Web Components with minimal overhead**
* Vite provides **extremely fast dev server and bundling**
* Together they create a **lightweight, framework-agnostic frontend setup**

## 📌 Notes

* No heavy framework runtime required
* Components are fully reusable across any framework or vanilla JS project
* Ideal for micro-frontends or design-system-driven architectures
